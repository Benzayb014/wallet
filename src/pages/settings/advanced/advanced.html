{{ 'Use Unconfirmed Funds' | translate }}
If enabled, wallets will also try to spend unconfirmed funds.
However, unconfirmed funds are not allowed for spending with merchants, BitPay Card loads, or BitPay in-app gift card purchases.
{{ 'Customize nonce' | translate }}
If enabled, the transaction nonce could be changed on the confirm view. This is an advanced feature, use cautiously.
{{ 'Enable BTC Replace-By-Fee' | translate }}
If enabled, a transaction designated as RBF could be accelerated using a higher fee.
Replace-By-Fee (RBF) is a node policy that allows an unconfirmed transaction in a mempool to be replaced with a different transaction that spends at least one of the same inputs and which pays a higher transaction fee.
